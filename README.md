# newmumps Rust Application

This is a Rust application initialized with Cargo.

## Getting Started

- Build the project:
  ```sh
  cargo build
  ```
- Run the project:
  ```sh
  cargo run
  ```

## Project Structure
- `src/main.rs`: Main entry point.
- `Cargo.toml`: Project manifest.

## Customization
Edit `src/main.rs` to start building your application.
